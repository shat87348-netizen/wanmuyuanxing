## 1. 项目身份

| 项 | 值 |
|---|---|
| 项目代号 | 数据处理软件 |
| 系统定位 | 后台数据处理任务的状态查看与配置管理 |

## 2. 主题（已实现，不动）

| 主题 ID | 名称 | 主色 | 中性色 | 圆角 |
|---|---|---|---|---|
| `military` | 军绿 | Emerald | Olive | 0 |
| `style1` | 风格 1 | Sky | Neutral | 0.25rem |
| `style2` | 风格 2 | Indigo | Gray | 0.125rem |

每个主题 × 浅 / 深 = 6 种组合。切换器在 Layout 右上角 `AppThemeMenu.vue`。

**约束**：
- 所有页面基于 CSS 变量（`--ui-primary` 等）配色，**不允许写死颜色**
- 新组件用军绿主题做主参考；其他主题靠变量自动适配
- 测试用 `tests/chrome/nuxtui-migration.chrome.mjs` 跑 6 主题回归

---

## 3. 告警 5 级配色

技术要求 4.2(2)e 多级预警的色阶：

| 级别 | 色名 | 视觉表现 |
|---|---|---|
| `INFO` | 蓝 | 普通通知 |
| `WARNING` | 黄 | 提醒 |
| `MINOR` | 橙 | 一般告警 |
| `MAJOR` | 红 | 重要告警 |
| `CRITICAL` | 暗红 | 紧急 |

**约定**：
- 颜色与图标即可，**不闪烁 / 不响铃 / 不弹屏**（不是值班系统）
- header 铃铛仅显示未读数；点击进告警中心
- CRITICAL 在浅色 / 深色主题下都要够对比度
- 红色专用于 MAJOR / CRITICAL；其他场景禁用红

|---|---|

| 兼容下限 | 1366×768 |
| 大屏 / 4K | 不要求 |
| 移动端 | 不要求 |

---

## 5. 字体

| 用途 | 字体 |
|---|---|
| 中文 | 系统默认（无需引入） |
| 数字 / 代号 | JetBrains Mono / 系统等宽 |
| 标题 | 加粗即可，不需要特殊处理 |

**不需要**：思源黑体、阿里巴巴普惠体等自定义字体（避免授权 / 体积问题）。

---

## 6. 图标

| 项 | 值 |
|---|---|
| 图标库 | `lucide` / `heroicons`（Nuxt UI 内置一致） |
| 风格 | 描边 stroke / 1.5px |
| 颜色 | 跟主题 `--ui-primary` 或 `currentColor` |
| 业务专有 | 无（不需要卫星/火箭等自绘图标） |

---

## 7. 数据可视化（ECharts）

| 项 | 值 |
|---|---|
| 主图表色 | 跟主题 `--ui-primary`（动态读取） |
| 多系列色 | 6 主题各定一组（最多 8 系列） |
| 网格 | `--ui-border` |
| 字号 | 12-14px |
| 工具栏 | 关掉 |
| 动画 | 关掉或 ≤ 300ms |

---

## 8. 交互动效

| 项 | 值 |
|---|---|
| 页面切换 | 无 / 极短淡入（< 150ms） |
| 模态 / 抽屉 | < 200ms |
| Toast | 右下角 / 3s 自动消失 |
| Hover | 颜色变化即可 |
| 数字滚动 | 不要 |
| 加载 | Skeleton 优先；Spinner 仅小区域用 |

---

## 9. 反模式（禁止）

- ❌ 玻璃拟态、毛玻璃、模糊
- ❌ 渐变背景（科技蓝 / 紫红 等）
- ❌ 大色块 + 装饰图
- ❌ 圆角过大（≤ 8px）
- ❌ 多重阴影（最多 1 层 subtle）
- ❌ 全屏视频 / 动效背景
- ❌ 告警闪烁 / 响铃 / 弹屏抢夺焦点
- ❌ 大屏 KPI 数字滚动动画
